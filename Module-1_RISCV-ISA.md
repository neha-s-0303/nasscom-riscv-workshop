# Module 1: Introduction to RISC-V ISA and GNU Compiler Toolchain

## Objective
This module introduces the fundamentals of the RISC-V Instruction Set Architecture (ISA) and the GNU compiler toolchain used for RISC-V software development.

## Topics Covered
- Introduction to RISC-V basic keywords
- Lab work for RISC-V software toolchain
- Integer number representation
- Signed and unsigned arithmetic operations

## Tools Used
- GCC (GNU Compiler Collection)
- RISC-V GNU Toolchain
- Spike RISC-V ISA Simulator
- GitHub Codespaces
